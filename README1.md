#Open CV 1주차

##1. 📌실습 개요

- 본 실습은 Python과 OpenCV를 이용하여 컴퓨터 비전의 기초적인 영상 처리 기능을 이해하고 구현하는 것을 목표로 한다.
- 컴퓨터 비전(Computer Vision)은 이미지나 영상으로부터 유용한 정보를 자동으로 추출하고 분석하는 기술로, 자율주행, 얼굴 인식, 객체 인식 등 다양한 분야에서 활용된다.
- 이번 실습에서는 Python 기반 영상 처리 라이브러리인 OpenCV 를 활용하여 이미지 읽기, 화면 출력, 마우스 이벤트 처리 등 기본적인 영상 처리 기능을 구현하였다.
